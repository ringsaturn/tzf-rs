## Usage

Add to `Cargo.toml`

```toml
tzf-rs = "0.1.2"
```

```rust
use tzf_rs::DefaultFinder;

fn main() {
    let finder = DefaultFinder::new();

    print!("{:?}\n", finder.get_tz_name(116.3883, 39.9289));
}
```
